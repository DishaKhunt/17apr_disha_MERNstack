3.How to create slider with animation?

1. slide Down:

➥ The jQuery slideDown() method is used to slide down an element.
Example:-
Script
                            $(document).ready(function(){
                                $('.down').click(function(){
                                    // $('.div1').slideDown();
                                    // $('.div1').slideDown('fast');
                                    // $('.div1').slideDown('slow');
                                    $('.div1').slideDown(1000);
                                });
                            });
