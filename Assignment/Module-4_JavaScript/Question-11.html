11. What are all the types of Pop up boxes available in JavaScript?

Answer:-
➥ There are three types of pop box.

  1. alert-box
  2. confirm-box
  3. prompt-box

1. Alert box:

➥ The alert function displays a pop-up dialog box with a specified message and an OK button. It is commonly used to show information to the user.
Example:-
JavaScript
                    function Useralert(){
                        alert("This is an alert box!");
                    }
Output:-

2. confirm box:

➥The confirm function displays a pop-up dialog box with a specified message, an OK button, and a Cancel button. It is used to get user confirmation for an action.
Example:-
JavaScript
                    function Userconfirom(){
                        var userConfirmed = confirm("Do you want to proceed?");
                    
                    if (userConfirmed) {
                        console.log("User clicked OK.");
                    } else {
                        console.log("User clicked Cancel.");
                    }
                    }
Output:-

3. prompt box:

➥ The prompt function displays a pop-up dialog box with a specified message, an input field for the user to enter data, and OK/Cancel buttons. It is used to get user input.
Example:-
JavaScript
                    function Userprompt(){
                        var userInput = prompt("Please enter your name:", "Disha Khunt");
                    
                    if (userInput !== null) {
                        alert("User entered: " + userInput);
                    } else {
                        alert("User clicked Cancel.");
                    }
                    }
