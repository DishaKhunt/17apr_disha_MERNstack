12. What is the function of the delete operator?

Answer:-
➥ The delete operator is used to delete properties from an object or elements from an array.
➥ It can also be used to delete variables, although this is less common.
Example:-

JavaScript
                        let emp = { 
                            firstName: "Raj", 
                            lastName: "Kumar", 
                            salary: 40000 
                        } 
                         
                        console.log(delete emp.salary);
                        console.log(emp);
