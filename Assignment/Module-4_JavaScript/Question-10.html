10. How can you convert the string of any base to an integer in JavaScript?

Answer:-
➥n JavaScript parseInt() function (or a method) is used to convert the passed-in string value to an integer value itself.
➥ This function returns an integer of the base which is specified in the second argument of the parseInt() function.
Example:-

JavaScript
                    var binaryString = "1101";
                    var decimalNumber = parseInt(binaryString, 2);
                    
                    console.log(decimalNumber);
