5. What are undeclared and undefined variables?

Answer:-

1. undeclared variable:

➥An undeclared variable is a variable that has been used in the code without being declared using a var, let, or const keyword.
➥ strict mode in JavaScript, using undeclared variables will result in an error.
Example:-
JavaScript
                                undeclaredVar = 25;
                                document.write(undeclaredVar);

2. undefined variable:

➥ An undefined variable is a variable that has been declared but not assigned a value, or it is used before any value is assigned to it.
➥ a variable is declared but not initialized, it automatically gets the value undefined by default.
Example:-
JavaScript
                                let undefinedVar;
                                console.log(undefinedVar);
