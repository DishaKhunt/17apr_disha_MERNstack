7. What is === operator?

Answer:-
➥ The === operator is a strict equality operator in JavaScript.
➥ The === operator compares both the value and the type of the operands.
➥ It returns true if the operands are of the same type and have the same value.
➥ It returns false if either the values or the types are different.
Example:-

JavaScript
                        var x = 5;
                        var y = "5";

                        document.write(x === y); 

                        var a = 10;
                        var b = 10;

                       document.write(a === b);
