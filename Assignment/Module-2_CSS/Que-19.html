19. What is the rule set?

Answer:-
➥The rule set means how to declare style in html documents and specific HTML element or group of elements should be styled on a web page or websites.
➥The rule set consists of a selector and a set of property-value pairs.
➥Basic structure of a CSS rule set :

css

            selector {
                property: value;
                /* More properties and values can be added */
              }

1. Selector :

➥ selector is used to target one or more HTML elements on a web page.
➥ There are many types of selectors.

  1. element Selector
  2. class Slector
  3. id Selector
  4. Child Selector
  5. Universal Selector
  6. Group Selector
  7. General-sibling Selector
  8. Adjsent-sibling Selector
  9. Psudo Slector :-
    • Psudo-class
    • Psudo-element

Example:-

css

                .header { 

                   /* Styles applied to all elements with class="header" */

                    }

2. property :

➥ The property represents a specific aspect of the element's presentation that you want to style.
➥Example of property are color, font-size, background-color, margin, padding, and many more.

2. values :

➥ The value is the specific value you want to assign to the property.

Example:-

css

                    h1 {
                        color: red;
                      }
➥In this rule, color is the property, and red is the value.
