5. What do you understand by CSS opacity?

Answer:-
➥ The opacity CSS property is set the opacity of an element.
➥ The Opacity describe the transpernacy of the element.
➥ 0% is completely transparent.
➥ 100% is default and represents the original image.

Example:-

✦ defult - 100% opacity ✦

✦ 50% opacity ✦

✦ 30% opacity ✦

✦ 0% opacity ✦
