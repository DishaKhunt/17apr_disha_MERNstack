18. What are the media types allowed by CSS?

Answer:-
➥There are many media types are allowed by CSS.

  1. all
  2. screen
  3. speech
  4. print

1. all:

➥ This is the default media type and applies to all devices.

Example:-

css

                                @media all {
                                    body {
                                      font-size: 12px;
                                    }
                                  }

2. screen:

➥This media type is used for computer screens, tablets, smartphones, and other similar devices with a screen.
➥It's the default for most web styles.

Example:-

css

                                @media screen {
                                    body {
                                      font-size: 12px;
                                    }
                                  }

3. speech:

➥This media type is used for screen readers and other speech synthesis devices.

Example:-

css

                                @media speech {
                                    body {
                                      font-size: 12px;
                                    }
                                  }

4. print:

➥This media type is for printed documents.
➥You can specify styles that should only be applied when a user prints a web page.

Example:-

css

                                @media print {
                                    body {
                                      font-size: 12px;
                                    }
                                  }
