7. How can image repetition of the backup be controlled?

Answer:-
➥ Image repetition of the background can be controlled using the background-repeat property in CSS.
➥ Syntax :-
background-repeat: repeat | repeat-x | repeat-y | no-repeat;

✦ Property Values ✦

Value Description
repeat The background image is repeated both vertically and horizontally.
repeat-x The background image is repeated only horizontally.
repeat-y The background image is repeated only vertically.
no-repeat The background-image is not repeated.
