6. How can the background color of an element be changed?

Answer:-
➥background-color property use the background color of an element be changed.
➥ The color can be set by:

• Name - specify a color name, like "red"
• HEX - specify a HEX value, like "#ff0000"
• RGB - specify a RGB value, like "rgb(255,0,0)"
• HSL - specify a HSL value, like "hsl(0, 100%, 50%)"
• hsla , rgba - specify the transparency , like "hsla(0, 255 , 0, 0.8)"

background-color: teal;

background-color: #026565;

background-color: rgb(4, 68, 68);

background-color: hsl(180, 92%, 10%);

background-color: rgba(4, 68, 68 , 0.6);

background-color: hsla(180, 92%, 10% , 0.3);
