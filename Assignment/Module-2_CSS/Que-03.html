3. What is the difference between CSS2 and CSS3?

Answer:-

No. CSS2 CSS3
1. Responsive designing is not supported in CSS2. Responsive designing is supported in CSS3.
2. CSS2 is very slow as compared to CSS3. CSS3 is faster than CSS2.
3. CSS2 does not support media queries. CSS3 supports media queries.
4. CSS codes are not supported by all types of browsers. CSS3 codes are supported by all browsers.
5. CSS2 is use the basic color only. CSS3 are use the HSL, RGBA, HSLA and grediant color.
6. CSS cannot produce 3D animation and transformation. CSS3 can produce 3D animation and transformation.
