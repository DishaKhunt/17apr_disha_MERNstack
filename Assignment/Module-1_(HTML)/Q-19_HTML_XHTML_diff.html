19. What is difference between HTML and XHTML?

Answer:-

No. HTML XHTML
1. HTML stands for Hypertext Markup Language. XHTML stands for Extensible Hypertext Markup Language.
2. It was developed by Tim Berners-Lee. It was developed by W3C i.e World Wide Web Consortium.
3. The format is a document file format. The format is a markup language.
4. All tags and attributes are not necessarily to be in lower or upper case. In this, every tag and attribute should be in lower case.
5. Filename extension used are .html, .htm. Filename extension are .xhtml, .xht, .xml.
