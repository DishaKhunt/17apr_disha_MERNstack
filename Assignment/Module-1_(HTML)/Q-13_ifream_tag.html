13. What is the use of an iframe tag?

Answer:-

=> The <iframe> tag specifies an inline frame.
=> The <iframe> tag in HTML is used to embed another HTML document within the current document.
=> It allows you to display content from another source, such as a website or web page, within the context of your own webpage.

1. Embed Path

➺ Syntax:-   <iframe src="url" title="description"></iframe>
➺ Example:-   <iframe src="Image/Image-Slider.html" height="700" width= 100% title="Iframe Example"></iframe>
