8. What are the various formatting tags in HTML?

Answer:-

❖ Formatting Tags :-

=>Formatting tags are used to change the formatting of text.
=>There are many types of the Formatting tags.

1. <b> tags:-

=>This tag is define the bold tag.
=>This tag is use to bold text.
➺ Syntax:- <b>......</b>
➺ Example:- <b>Disha</b>
➺ Output:- Disha

2. <strong> tag:-

=>This tag define the strong tag.
=>This tag is use to bold text.
➺ Syntax:-   <strong>......</strong>
➺ Example:- <strong>Disha</strong>
➺ Output:- Disha

3. <i> tag:-

=>This tag define the Italic tag.
=>This tag is use to italic text.
➺ Syntax:- <i>......</i>
➺ Example:- <i>Disha</i>
➺ Output:- Disha

4. <em> tag:-

=>This tag define the Emphasis tag.
=>This tag is use to italic text.
➺ Syntax:- <em>......</em>
➺ Example:- <em>Disha</em>
➺ Output:- Disha

5. <u> tag:-

=>This tag define the Underline tag.
=>This tag is use to underlin in the text.
➺ Syntax:- <u>......</u>
➺ Example:- <u>Disha</u>
➺ Output:- Disha

6. <mark> tag:-

=>This tag define the Mrak tag.
=>This tag is use to Hylight(mark) text.
➺ Syntax:- <mark>......</mark>
➺ Example:- <mark>Disha</mark>
➺ Output:- Disha

7. <del> tag:-

=>This tag define the Delet tag.
=>This tag is use to delet text.
➺ Syntax:- <del>......</del>
➺ Example:- <del>Disha</del>
➺ Output:- Disha

8. <strike> tag:-

=>This tag define the Strike tag.

9. <q> tag:-

=>This tag is use to quoted text.
➺ Syntax:- <q>......</q>
➺ Example:- <q>Disha</q>
➺ Output:- Disha

10. <blockquote> tag:-

=>This tag define the BlockQuote tag.
=>This tag is use to block level quotation.
➺ Syntax:- <blockquote>......</blockquote>
➺ Example:- <blockquote>Lorem ipsum dolor sit amet consectetur adipisicing elit. Veritatis ullam aperiam saepe et ad fugiat voluptatibus possimus eaque, a dolores? Dicta possimus ipsum sunt architecto repudiandae? Rerum vero enim ea.</blockquote>

11. <sub> tag:-

=>This tag define the Subscript tag.
=>This tag is use to add the Subscript text.
=>Subscript text appears half a character below the normal line and is sometimes rendered in a smaller font. ➺ Syntax:- <sub>......</sub>
➺ Example:- H<sub>2</sub>O
➺ Output:- H2O

12. <sup> tag:-

=>This tag define the Superscript tag.
=>This tag is use to add the Superscript text.
=> Superscript text appears half a character above the normal line and is sometimes rendered in a smaller font. ➺ Syntax:- <sup>......</sup>
➺ Example:- H<sup>2</sup>
➺ Output:- H2

13. <p> tag:-

➺ Syntax:- <p>......</p>
