7. What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements?

Answer:-

'id' attribute 'class' attrinute
id attrinute is unique. class attribute is not unique.
id attribute is used to assign only one id name to an HTML element. class attribute is used to assign one or more class names to an HTML element.
id attribute use in CSS symbole is "#". class attribute use in CSS symbole is ".".
➺ Example:-
                        file.html 
                        <div id="xyz"> This is a div tag. </div>
                        
                        file.css 
                        #xyz{
                            color: red;
                        }
➺ Example:-
                        file.html
                        <div class="d1">
                        <p class="p1">This is a paregraph.</p>
                        </div>
