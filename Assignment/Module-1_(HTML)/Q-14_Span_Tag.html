14. What is the use of a span tag? Explain with example?

Answer:-

=> <span> tag is an inline container used to mark up a part of a text, or a part of a document.
=> <span> tag is easily styled by CSS using the class or id attribute.
=> span tag is paired tag means it has both open tag and close tag. => The <span> tag is used for the grouping of inline elements.
=> span is very similar to the div tag, but div is a block-level tag and span is an inline tag.

➺ Syntax:- <span class="">Some Text</span>
➺ Example:-

        <h3>Itroduction</h3>
        <p>
            My name is <span style="color:red;font-weight:bolder">Disha.</span> <br>
            I am a <span style="background-color: lightgreen;">Computer Engineering</span> Student.
        </p>
➺ Output:-

Itroduction

My name is Disha.
I am a Computer Engineering Student.
