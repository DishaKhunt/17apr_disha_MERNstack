10. How can we club two or more rows or columns into a single row or column in an HTML table?

Answer:-

=>Two or more row or column into a single row or column in used to a 'rowspan' and 'colspan' attribute.

1. Combine row:

=> Combine row in used to 'rowspan' attribute on a <td> or <th> element to specify the number of rows it should span.
➺ Example:-
            <table>
                <tr>
                  <td rowspan="2">Name</td>
                  <td>Disha</td>
                </tr>
                <tr>
                  <td>Isha</td>
                </tr>
              </table>
Name Disha
Isha

2. Combine Column:

=> Combine Column is used to 'colspan' attribute on a <td> or <th> element to specify the number of columns it should span.
➺ Example:-
            <table>
                <tr>
                  <td colspan="2">Name</td>
                </tr>
                <tr>
                  <td>Disha</td>
                  <td>Isha</td>
                </tr>
              </table>
Name
Disha isha
