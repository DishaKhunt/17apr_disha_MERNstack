6. What is the ‘class’ attribute in HTML?

Answer:-

=> Class attribute is used to assign one or more class names to HTML element.
=> Class attribute can be used to style and manupliate elements using in CSS.
➺ Example:-

            file.html
            <div class="d1">
            <p class="p1">This is a paregraph.</p>
            </div>

            file.css 
            .div{
                background-color: #000;
            }
            .p{
                padding: 10px;
            }
