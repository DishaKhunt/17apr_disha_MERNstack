17. What are the different tags to separate sections of text?

Answer:-

=>There are many tags use to separate and structure different sections of text within a document.

1. <h1> - <h6> : Heading Tags:-

=> Heading tags are used to define headings or section titles.
=> They represent different levels of headings, with <h1> being the highest and <h6> being the lowest.
➺ Syntax:-
      <h1>......</h1>
      <h2>......</h2>
      <h3>......</h3>
      <h4>......</h4>
      <h5>......</h5>
      <h6>......</h6>

➺ Example:-
      <h1>Disha</h1>
      <h2>Disha</h2>
      <h3>Disha</h3>
      <h4>Disha</h4>
      <h5>Disha</h5>
      <h6>Disha</h6>

➺ Output:-

Disha

Disha

Disha

Disha

Disha
Disha

2. <p>: Paragraph Tag:-

=> The <p> tag is used to define a paragraph of text.
➺ Syntax:- <p>......</p>
➺ Example:- <p>Lorem ipsum dolor sit, amet consectetur adipisicing elit.</p>
➺ Output:-

Lorem ipsum dolor sit, amet consectetur adipisicing elit.



3. <div> : Division Tag:-

=> <div> tag commonly used to group and separate sections of content.
=> This tag used css styling.
➺ Syntax:- <div>......</div>
➺ Example:- 
        <div class="section">
            <h3> Title</h3>
            <p>This is paregraph tag.</p>
          </div>
➺ Output:-

Title

This is paregraph tag.


4. <section> : Section Tag:-

=> Section tag is used to distribute the content i.e., it distributes the sections and subsections.
➺ Syntax:- <section>......</section>
➺ Example:- 
            <section>
            <h3>Section</h3>
            <p>This is the content of section.</p>
➺ Output:-

Section

This is the content of section.
