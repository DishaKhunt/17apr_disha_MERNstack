1. Are the HTML tags and elements the same thing?

Answer:-

=> No, HTML tags and elements are not the same thing.
  Beacuse..............

=> HTML tags are used to tell the browser how to show the content.
=> There are two types of tags.
=> 1.pair tag
      for example:- <html></html>
=> 2.self tag (void tag , unpair tag)
     for example :- <br>

=> HTML elements are combination of an opening tag, content, and closing tag.
=> for Example :- <body>this is a body tag.</body>
       - <body> is a opening tag.
       - "this is a body tag." is a content.
       - </body> is a closing tag.
