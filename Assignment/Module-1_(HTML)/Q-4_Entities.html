4. What are HTML Entities?

Answer:-

=> Entities are used replacement of reserved characters in HTML.

=> for example:-
  → &lt;(<) = represent the less than
  → &gt;(>) = represent the grater than
  → &#128527;(😏) = represent the smirking face emoji
  → &#128522;(😊) = represent the smiling face with smiling eyes emoji
