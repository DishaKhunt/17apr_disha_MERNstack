5. What is the difference between Bootstrap 4 and Bootstrap 5

BASIS OF BOOTSTRAP 4 BOOTSTRAP 5
Grid System It has 5 tier (xs, sm, md, lg, xl). It has 6 tier (xs, sm, md, lg, xl, xxl).
Color It has limited colors. Extra colors added with the looks,
Icons Does not include a default icon library. Integrates the Bootstrap Icons library by default, providing a set of scalable SVG icons.
Form Control Styling Has form control styling based on browser defaults. Provides more consistent form control styling across different browsers.
