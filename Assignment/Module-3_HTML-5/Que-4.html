4. Canvas and SVG tags

Answer:

✦ Canvas:-

➥Canvas is also use to create the graphics.
➥Canvas is a raster base graphics.
➥To create canvas you need to learn the basic javascript.
➥Canvas is blur on the zoom because it's use the raster base graphics.
➥Canvas did use pixel.
➥Can create the canvas using 'canvas' tag(using javascript).
Example:- 

                    <canvas id="can" width="500px" height="250px">
                        Canvas isn't work properly
                    </canvas>
                
                    <script>
                        var con = document.getElementById('can');
                        var co = con.getContext('2d');
                        console.log(co);
                        co.fillStyle  = '#c90e4c';
                        co.fillRect(60, 50, 150, 150);
                    </script>
Output:-
Canvas isn't work properly

✦ Svg:-

➥Svg is stands for the Scalable Vector Graphics
➥ is use to create the graphics.
➥ is a vector base graphics.
➥To create SVG no need to learn the javascript.
➥ isn't blur on the zoom because it's use the vector base graphics.
➥ didn't use pixel.
➥Can create the SVG using 'svg' tag.
Example:- 

            <svg height="500px" width="500px" style="border: 1px solid red;">
                 <circle cx="200" cy="200" r="100" fill="lightblue" 
                 stroke="blue" stroke-width="10"/>
            </svg>
