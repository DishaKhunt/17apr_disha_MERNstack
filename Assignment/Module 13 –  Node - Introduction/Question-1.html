1. What is the difference between JS and Node.js

Aspect JavaScript (JS) Node.js
Environment Runs in web browsers (client-side). Runs on the server (server-side) using a runtime environment.
Execution Context Works primarily for interacting with the DOM and web APIs. Works with system resources, databases, and can handle HTTP requests.
Use Case Used for front-end development to enhance user interaction. Used for back-end development, handling APIs and servers.
Libraries/Frameworks React, Angular, Vue.js for front-end. Express.js, built-in modules (http, fs) for back-end development.
Asynchronous Handling Supports async programming, but mostly for client-side tasks. Highly optimized for handling async tasks, especially I/O operations.
