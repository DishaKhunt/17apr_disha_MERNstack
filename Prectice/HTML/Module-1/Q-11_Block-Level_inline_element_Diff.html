11. What is the difference between a block-level element and an inline element?
Block-level Element Inline Element
Block Level element is always start of new line. Inline element is dose not start the new line.
Block-level element always takes up the full width available. Inline element only takes up as much width as necessary.
Example:- <p> , <pre> , <div> Example:- <b> , <i> , <u>
