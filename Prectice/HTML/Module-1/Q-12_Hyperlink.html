12. How to create a Hyperlink in HTML?

Answer:-

❖ Hyper link :-

=>Hyper link creat to HTML in the use to the <a> tag.
=><a> tag is define the anchore tag.

➺ Syntax :-
=><a href="URL">Link text </a>
=> <a> :- this is a anchore tag.this used the creat the hyperlink.
=> herf="URL" :- herf is attribute. this is specifies the URL.
=> Link text :- This is the text that will be displayed as the hyperlink.

➺ Example :-
=> <a href="https://www.google.com">Google </a>

=> Hyper link open the new brower tab or window by adding the target attribute to the <a> tag.
➺ Example :-
=> <a href="https://www.amazon.in" target="_blank" >Amazon</a>
➺ Output :- Amazon
