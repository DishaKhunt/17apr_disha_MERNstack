5. What are different types of lists in HTML?

Answer:-

❖ HTML list

➺ Types of list

=> There are three types of HTML lists.
  1. Unordered List
  2. Orderded List
  3. Data List

1. Unordered List

=>Unorderd list used to when a set of items can be placed in any order.

➺ Syntax:-
    <ul></ul> - Unordered list
    <li></li> - List item

➺ Example:-
=>Syntax:- 
        <h4> Car Brand Name </h4>
        <ul> 
              <li> Toyota </li>
              <li> Audi </li>
              <li> Jaguar </li>
              <li> Mercedes-Benz </li>
        </ul>
=>Output:-

Car Brand Name

  • Toyota
  • Audi
  • Jaguar
  • Mercedes-Benz

➺Attribute of Unoreded List

1) Syntax:-
  <ul type = "square"> <li></li> </ul>

=> Example:-
  • Pen
  • Pencile
2) Syntax:-
  <ul type = "circle"> <li></li> </ul>

=> Example:-
  • Pen
  • Pencile
3) Syntax:-
  <ul type = "disc"> <li></li> </ul>

=> Example:-
  • Pen
  • Pencile

2. Ordered List

=>Ordered are used to display a list of items that should be in a specific order.

➺ Syntax:-
    <ol></ol> - Ordered list
    <li></li> - List item

➺ Example:-
=>Syntax:- 
        <h4> Food Name </h4>
        <ol> 
              <li> Dhosa </li>
              <li> Pizza </li>
              <li> Burger </li>
              <li> Sendwitch </li>
        </ol>
=>Output:-

Food Name

  1. Dhosa
  2. pizza
  3. Burger
  4. Sendwitch

➺Attribute of Oreded List

1) Syntax:-
  <ol type = "1"> <li></li> </ol>

=> Example:-
  1. Pen
  2. Pencile
2) Syntax:-
  <ol type = "A"> <li></li> </ol>

=> Example:-
  1. Pen
  2. Pencile
3) Syntax:-
  <ol type = "a"> <li></li> </ol>

=> Example:-
  1. Pen
  2. Pencile
4) Syntax:-
  <ol type = "I" start="4"> <li></li> </ol>

=> Example:-
  1. Pen
  2. Pencile

3. Data List

=>It's used to display data and their definition.

➺ Syntax:-
    <dl></dl> - Data list
    <dt></dt> - Data term
    <dd></dd> - Data Definition

➺ Example:-
=>Syntax:- 
            <dl>
                <dt>HTML</dt>
                <dd>- Hyper Text Markup Language. It is use to create the structure of the web page. </dd>
            </dl>

=>Output
HTML
- Hyper Text Markup Language. It is use to create the structure of the web page.
