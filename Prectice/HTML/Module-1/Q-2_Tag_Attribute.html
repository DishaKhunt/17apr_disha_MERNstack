2. What are tags and attributes in HTML?

Answer:-

❖ HTML tag:-

=> HTML tags are used to tell the browser how to show the content.
=> There are two types of tags.
=> 1.pair tag
      for example:- <strong></strong>
=> 2.self tag (void tag , unpair tag, empty tag)
     for example :- <image>

❖ HTML attribute:-

=> Attribute is use to define the property of the element.
=> Attribute is use to define additional information.
=> Attribute is compulsory define inside the start tag.
=> Example:-<html lang="en"><html>
