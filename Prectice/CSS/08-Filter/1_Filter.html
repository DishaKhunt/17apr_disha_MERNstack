Filter

=> The filter property defines visual effects (like blur and saturation) to an element (often <img>).
=> Syntax :-
filter: none | blur() | brightness() | contrast() | grayscale() | hue-rotate() | invert() | opacity() | saturate() | sepia() | url();

✦ Property Values ✦

Value Description Example
blur(px) Applies a blur effect to the image. A larger value will create more blur. Demo❯
brightness(%) Adjusts the brightness of the image. Demo❯
grayscale(%) Converts the image to grayscale. Demo❯
saturate(%) Saturates the image. Demo❯
sepoia(%) Converts the image to sepia. Demo❯
opacity(%) Sets the opacity level for the image. The opacity-level describes the transparency-level. Demo❯
invert(%) Inverts the samples in the image. Demo❯
hue-rotate(deg) Applies a hue rotation on the image. Demo❯
url() The url() function takes the location of an XML file.
